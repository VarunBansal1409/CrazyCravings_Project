# CrazyCravings
"CrazyCraving is a full-stack food delivery web application built with React on the frontend and Java Spring Boot on the backend, designed to offer fast browsing, smooth ordering, and a clean user experience. The platform integrates secure APIs, efficient state management, and modular components to ensure reliability and scalability. It is deployed using Vercel for the frontend and Render for backend services, resulting in seamless performance and easy accessibility. Explore the live website here: https://crazycravings.vercel.app/
.

This project demonstrates streamlined full-stack development with a focus on simplicity, performance, and clean architecture, making it easy to maintain, extend, and adapt as new features or improvements are introduced, while ensuring a smooth workflow across both client and server components."
